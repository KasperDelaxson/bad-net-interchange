NNM Setup

Welcome to Network Node Monitor (NNM)

This NNM compares latest IGM grid scenarios against CGMA in-house net positions. To do that, the browser needs read access to two network drives.

Getting Started:

  1. Use Chrome or Edge — access granting is not available in Firefox or Safari.
  2. For each drive below, follow the three steps to grant read access (see the How to grant access box inside each card).
  3. Once both shares show ✓ Granted, the comparison runs automatically.
  4. Use the Diff, Compare, and Tables tabs to view results.

\\fs61\driftdata\Drift\Arkiv\CGMES\OFFLINE\

How to grant access
  1. Click Copy to put the path above on your clipboard.
  2. Click Grant IGM Root. A folder picker opens.
  3. Paste the path into the picker's address bar, press Enter, then click the confirm button — usually Select Folder (Chrome / Edge on Windows), but it may read Open, Choose or Select depending on your browser and OS.
  4. If a prompt appears asking whether the system can access the file paths, select Allow on every visit (Tillad ved hvert besøg).

Troubleshooting
The picker won't accept the path.
Make sure the share is mounted in Windows Explorer first (open it once in File Explorer). Then paste the path into the picker's address bar and press Enter before clicking the confirm button. If not on Energinet Prem, use VPN.
I don't see a Grant button, or nothing happens when I click it.
The folder picker is only available in Chromium-based browsers (Chrome, Edge). Open the page in Chrome or Edge instead of Firefox or Safari.
I'm being asked to grant access again on a new visit.
The browser may have cleared site data or revoked the permission. Just repeat the grant steps — nothing is broken and previous results are not lost.
The comparison doesn't start after I grant both shares.
Check that both rows above show ✓ Granted. If one still says Pending, click its Grant button again. You can also enable debug logging below and check the browser console for errors.
